# Paper

Paper is a platform for buying and selling products.

## Tech Stack

- Next.js
- TailwindCSS

## How to run

But first, install the dependencies.

```bash
npm install
```

Then, run the development server.

```bash
npm run dev
```

## How to build

```bash
npm run build
```
